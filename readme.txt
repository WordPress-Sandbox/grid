Allows to create complex Centerpages, composed of Widgets, Plain Text and Post-References

== Description ==

The Grid gives you the ability to add complex grid-Layouts to any post or page.
These grids consist of 1-n regions, which could hold 1-n containers.
Each container could contain

* A Widget
* Plain Text/HTML
* A reference / teaser to a post or page or anyother thing in the post table

Development takes place in this Test-Site:
http://argo.exmachina.ws/
